Programming Fundamentals

Comprehensive Course Outline

Course Modules

1

Overview of Computer and Programming

  • Introduction to computers (then and now)
  • Computer hardware components
  • Software categories and types
  • The software development methodology
  • Programming paradigms overview
2

Designing Algorithm I

  • Introduction to algorithms
  • Steps in creating effective algorithms
  • Using NS Diagrams for visualization
  • Decision tables for complex logic
Lab: Designing NS Diagrams and decision tables using tools
3

Designing Algorithm II

  • Designing algorithms with pseudocode
  • Flowchart creation techniques
  • Selection structures in algorithms
  • Repetition structures and loops
Lab: Designing flowcharts using drawing tools
4

Overview of Java

  • Java language elements and syntax
  • Variable declarations and scope
  • Data types and type systems
  • Executable statements and structure
Lab: Introduction to Java and running a simple program
5

Data Types, Operators, and Simple Functions

  • Understanding primitive data types
  • Arithmetic expressions and precedence
  • Writing mathematical formulas in Java
  • Using standard library functions
  • Basic function creation and invocation
6

Selection and Repetitive Structures

  • Implementing selection structures (if-else)
  • Switch statements for multiple branches
  • While loops for conditional repetition
  • For loops for counted iterations
  • Loop control statements (break, continue)
